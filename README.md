# Becoming-rustacean
7 CLI tools, to become fluent in Rust Programming Language. Starting from easy to difficult.
this will all be re-implementing some unix tools, and documenting my experience good/bad and how
each completed project had changed my aproach if any, as a programmer.

### List of cli tools to Re-implement.
 - Echo 
 - Cat
 - Head
 - Find
 - Grep
 - Curl
 - Git
check out the [wiki](https://github.com/komilazio/Becoming-rustacean/wiki) to see what i learned working on these 7 projects.
